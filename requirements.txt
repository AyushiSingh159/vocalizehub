Flask==2.2.3

Explanation of Dependencies:
Flask: A lightweight Python web framework used to build the web application.

pyttsx3: A text-to-speech conversion library that works offline and supports multiple platforms (Windows, MacOS, Linux).

gTTS: A Python library and wrapper for Google Text-to-Speech API, used for generating speech from text.

google-cloud-texttospeech: A library for using Google Cloud's Text-to-Speech API to generate high-quality, natural-sounding speech.

How to Use requirements.txt:
Save the above content into a file named requirements.txt in your project directory.

Install the dependencies using pip:

bash
pip install -r requirements.txt
This will ensure all necessary libraries are installed for running the VocalizeHub project. If additional libraries are required later, they can be added to this file and reinstalled using the same command.
